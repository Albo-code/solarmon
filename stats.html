stats

Usage

usage: stats.py [-h] data_file

Read engery data from supplied `.json` file (created using `energy` script)
and output various statistics.

positional arguments:
  data_file   .json file containing energy data

options:
  -h, --help  show this help message and exit

Function descriptions

Script using various descriptive statistical methods for analysing solar energy data.

Script accepts .json file, as output by energy script.

stats.get_energy_dataframe(df: pandas.DataFrame) → Tuple[pandas.DataFrame, str, str][source]

From the given energy pandas dataframe extract the timeUnit, unit and values data.

Returns

Tuple containing 3 items: [1] Pandas dataframe containing the energy values, [2] Time unit of the energy data, [3] Energy value units

Return type

Tuple[pd.DataFrame, str, str]
